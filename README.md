# sixmwd.com
Ionic Project (Angular). Intent to be a Web app.

## What you need
- NodeJS
- Ionic@4

## Build && Deploy
Build with
```
ionic build --prod
```
Deployed via Netlify: Just push from master branch to deploy changes
```
git push
```
